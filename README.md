# Deep Research Agent – Educacional

Sistema agencial de pesquisa profunda voltado para **educação**, atuando como:
- Tutor inteligente
- Orientador de pesquisa
- Avaliador formativo

## Foco Educacional
- Aprendizagem guiada
- Feedback explicativo
- Níveis cognitivos (Bloom)
- Evita respostas prontas

## Execução
```bash
pip install -r requirements.txt
python main.py
```
